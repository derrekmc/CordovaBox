This is a websocket based chat server in Node.jS

Installation notes:
1) Install NVM
2) run command below to install dependencies:
   $ npm install
3) run server:
   $ npm start

Install NVM ( Node Version Manager ):
curl https://raw.github.com/creationix/nvm/master/install.sh | sh
